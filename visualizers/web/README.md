Web Visualizers
===============

Installing Node
---------------
- Install `Node` and `npm`
- `cd` to `/visualizers/web/node`
- Enter `npm init` to use a wizard to create the package.

Installing Dependencies
-----------------------
- `cd` to `/visualizers/web/node`
- Enter `npm install`

Start Node Web App
------------------
- `cd` to `/visualizers/web/node`
- Enter `node app.js`
- Site should now be hosted from local.
- Navigate to main page at `index.html`.
